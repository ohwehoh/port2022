ABOUT

안녕하세요? 저는 프론트엔드 개발자를 꿈꾸는 이연우입니다. 컴퓨터공학을 전공하고, 2019년에 졸업하였습니다. 적성에 맞는 직업을 찾고자 먼 길을 돌아 2021년 말 부터 프론트엔드 학원을 다니기 시작하였고 이제 정착할 곳을 찾았습니다.

6개월 동안 기초부터 차근차근 공부하다보니 머릿속에 흐릿하게 떠다니던 지식들이 정리되면서 웹 개발 공부에 더욱 흥미가 생기고 자신감도 생겼습니다. 그리고 프로젝트를 할 때 기초 지식에 살을 붙여가며 제가 생각한 기능들을 하나씩 구현하데 큰 재미를 느껴 이 분야로 평생 먹고 살고 싶다는 생각이 들었습니다.

긴 취업 준비 생활로 자신감이 많이 하락 한 상태였는데 교육을 받으며 열정적으로 변한 스스로에게 자신감이 생겼습니다. 이 자신감을 바탕으로 더욱 자신감을 잃지 않게 노력하는 프론트엔드 개발자가 되고싶습니다.

SKILL

HTML

웹표준과 웹접근성을 준수한 마크업을 할수 있으며 깔끔한 코드 정리를 통해 누가봐도 읽기 쉬운 코드로 작성이 가능합니다. 최근 HTML5 시멘틱 태그를 용도에 맞게 적재적소 사용하여 하드코딩이 가능하며 크로스 브라우징에 대해 이해하고 있습니다.

CSS

CSS의 각종 속성을 적절하게 사용하며 IR기법이나 이미지 스프라이트등과 같은 최신의 CSS트렌드를 항상 주의깊게 지켜보고 있습니다. 트랜지션과 트랜스폼, 키프레임을 이용해 간단한 애니메이션을 만들 수 있습니다. 그리고 항상 코드의 재활용에 대해 고민합니다.

JAVASCRIPT

슬라이드나 갤러리와 같은 플러그인의 옵션을 잘 활용하고 디자인에 맞게 적용 할 수 있습니다. 탭메뉴나 레이어팝업과 같은 페이지를 구성할 때 자주쓰이는 것들을 스크립트를 사용하여 구현합니다.

JQUERY

스크립트가 데이터베이스와 연동이 필요할 때 주로 사용하였습니다. 구현에 필요한 예제를 보고 응용하여 작업 할 수 있습니다. 제이쿼리로 게시판의 좋아요, 회원가입 유효성 검사를 구현한 경험이 있습니다.

PHP

PHP 프로젝트 수업을 듣고 응용하여 팀 프로젝트에 적용하면서 익혔습니다. PHP를 활용하여 온라인 게시판/블로그 등의 기능을 구현한 경험이 있습니다.

MYSQL

php사이트 만들기 수업을 듣고 팀 프로젝트에 적용하면서 데이터베이스의 테이블에서 원하는 데이터를 조회하고 수정하고 삭제 할 수 있는 쿼리문을 작성할 수 있게 되었습니다. 현재 SQLD자격증을 공부중입니다.

REACT

REACT를 통하여 프레임워크를 사용해봤습니다. 싱글페이지의 구성과 데이터의 흐름을 완벽하게 이해하고 이를 응용하여 react 사이트를 만든 경험이 있습니다.
